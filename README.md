为什么有这本教程
=======

本书作为在开智学院编程课程学习文档书籍 记录自己的学习 经历 体验

学习是个积累的过程 踏踏实实地走 路上难免跌倒 入坑   
记录自己的学习过程 是一个输出的过程 是巩固的过程 是自我反馈激励的过程  
所以有了这本书 持续成长 看见成长的自己

- 明确一点 应该是**教程** 能教会6个月前 那个不会的自己 `*`增补
